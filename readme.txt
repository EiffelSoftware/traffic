1. General
____________________________________________________________

Traffic is a library that comes with three example applications. To use the Traffic 
library in your own projects, just include the library directory in your ace file. 
The current version of Traffic uses EiffelMedia for visualization.

2. Requirements
____________________________________________________________

General:
	- Eiffel Studio 5.6 for Linux and Windows
	- EiffelMedia 0.7.0

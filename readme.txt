1. General
____________________________________________________________

Traffic is a library that comes with many example applications. To use the Traffic 
library in your own projects, just include the library directory in your ace file. 
The current version of Traffic uses EiffelMedia for visualization.

2. Requirements
____________________________________________________________

General:
	- Eiffel Studio 5.6 for Linux and Windows
	- EiffelMedia 0.8.0

3. Building an example
____________________________________________________________

Traffic relies on the gobo tools to generate ace-files for linux/windows.
To compile one of the examples open a command prompt or shell in in the directory 
"example/" or any of its subdirectories where an example application is located,
and type "geant install_develop" (includes some assertions) or "geant install" 
(no assertions). Doing this will provide you with an ace-file (called either 
ise.ace oder developer.ace) that is tailored to your system. Open this ace-file
in EiffelStudio by clicking "File -> New Project" and "Open existing ace-file". EiffelStudio
will generate a new project, compile it for you, and you will be ready to go!

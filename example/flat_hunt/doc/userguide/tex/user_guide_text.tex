  \paragraph{}
  \emph{Flat Hunt} is an application that is used to teach you programming, along with another application named \emph{Touch} \cite{ugrk05}. \emph{Flat Hunt} is a "Scotland Yard"-like game that will mainly appear in the assignments for the Introduction to Programming course. It is based on \emph{TRAFFIC} \cite{ugsa05} for modeling the city where the game takes place and on \emph{EiffelMedia} (formerly known as \emph{ESDL} \cite{ugtgb03}) for visualization.\linebreak[2]

  This document describes how to use \emph{Flat Hunt}.
